# AI_USAGE_ANDRES_MORA

# Herramienta de IA utilizada

En el trancurso del desarrollo utilicé ChatGPT como apoyo principal para estructurar la solución o desarrollo de la prueba. Fue una herramienta muy útil para avanzar más rápido, aunque siempre necesitó revisión y ajustes manuales para que el resultado fuera realmente funcional.

# Prompts utilizados
1. "Crea una mini aplicación en React con Express para gestionar tareas, con endpoints CRUD y una interfaz sencilla."
2. "Añade validaciones, manejo de errores y un diseño responsivo (responsive design) para la vista de tareas."
4. "Ayúdame a organizar el código para que sea más claro, ordenado y fácil de mantener."

# Partes generadas con IA
La IA me ayudó a arrancar con la estructura general del proyecto, especialmente en la parte del frontend y del backend. También colaboró en la lógica para crear, listar, actualizar y eliminar tareas, además de aportar una base para los estilos y la organización visual de la aplicación.

# Partes corregidas manualmente
Aunque la IA aportó una base sólida, fue necesario corregir varios detalles para que todo encajara mejor. Ajusté la interfaz dado que se me da muy facil html y css, revisé los endpoints y les hice pruebas con Postman para asegurar que respondieran correctamente dado que generaban errores a la hora de obtener los datos del Json y adapté parte del contenido generado para que coincidiera mejor con los requisitos de la prueba técnica. También validé manualmente el comportamiento completo de la aplicación para comprobar que funcionara de forma real.

# Aprendizaje
En esta prueba reforcé algunas cosas que ya sabia, como que se puede acelerar mucho el desarrollo de soluciones de programacion, pero no reemplaza la vision que tiene el programador de la app. Reforce el ruteo de endpoint, codigos de erros, formato json y los limites entre frontend y backend y a tener presente que se debe revisar críticamente lo que genera la IA para que el resultado sea acorde con los requisitos que se realizaron en el levantamiento.

# Nota personal
Fue necesario indicarle a la IA que usara React para el frontend y Express para el backend, porque la prueba pedía una solución práctica con tecnologías JavaScript claras y fáciles de ejecutar y por demas eso lo facilita express. En general, trabajar con IA me permitió avanzar con más confianza, aunque siempre fue importante terminar de pulir el resultado a mano.
